# Abby's World – Season 2

This guide walks you through installing **Abby's World – Season 2** using your preferred method.  
Choose **one** of the options below and follow the steps carefully.
HEADS UP! DO NOT ENABLE OR UPDATE ANY OF THESE MODS UNLESS I UPDATE THE MODPACK!!

## Modrinth Installation

> Recommended if you use the Modrinth App.

### Steps
1. Download and install the **Modrinth App**
2. Open the app and go to **Create Instance**
3. Choose the correct Minecraft version
4. Import the Abby's World – Season 2 pack
5. Launch the instance and verify it loads correctly

## CurseForge Installation

> Recommended if you use the CurseForge launcher.

### Steps
1. Download and install the **CurseForge App**
2. Open Minecraft in the launcher
3. Click **import**
4. Select **zip file**
5. Upload the zip
6. Import the Abby's World – Season 2 pack
7. Launch and confirm everything loads properly

## Base Fabric (Manual Installation)

> If your weird and dont have a launcher

### Requirements
- Minecraft (1.20.1)
- Fabric Loader
- Fabric API

### Steps
1. Install **Fabric Loader** for the correct Minecraft version
2. Launch Minecraft once with Fabric, then close it
3. Open your `.minecraft/mods` folder
4. Place all mod files into the `mods` folder
5. Ensure **Fabric API** is installed
6. Launch Minecraft using the Fabric profile
